how many type of JS Event? How to use it ? There are many type of events in JavaScript some of them are listed below : click: Triggered when a mouse button is clicked. dblclick: Triggered when a mouse button is double-clicked. mousedown: Triggered when a mouse button is pressed. mouseup: Triggered when a mouse button is released. mousemove: Triggered when the mouse pointer is moved. mouseover: Triggered when the mouse pointer moves over an element. mouseout: Triggered when the mouse pointer moves out of an element. keydown: Triggered when a key is pressed down. keypress: Triggered when a key is pressed and released
